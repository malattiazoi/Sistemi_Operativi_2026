       which  match the selection criteria to stdout.  All the criteria have to match.

              $ pgrep -u root sshd

       pidwait will wait for each process instead of listing them on stdout.

       -signal
       --signal signal
              Defines  the signal to send to each matched process.  Either the numeric   
              or the symbolic signal name can be used.  (pkill only.)

       -c, --count
              Suppress normal output; instead print a  count  of  matching  processes.

              signaled or waited for.

       -d, --delimiter delimiter
              Sets the string used to delimit each process ID in the  output  (by  de‐

              Display name and PID of the process being killed.  (pkill only.)

              The  pattern is normally only matched against the process name.  When -f

              is translated into pgrep's, pkill's, or pidwait's own process group.

              Only  match processes whose real group ID is listed.  Either the numeri‐   
              cal or symbolical value may be used.

       -i, --ignore-case
              Match processes case-insensitively.

              List the full command line as well as the process ID.  (pgrep only.)

              Only match processes whose parent process ID is listed.

       -s, --session sid,...
              Only match processes whose process session ID is listed.  Session  ID  0   
              is translated into pgrep's, pkill's, or pidwait's own session ID.

       -t, --terminal term,...
              Only match processes whose controlling terminal is listed.  The terminal   
              name should be specified without the "/dev/" prefix.

       -u, --euid euid,...
              Only match processes whose effective user ID is listed.  Either the  nu‐

       -U, --uid uid,...
              Only match processes whose real user ID is listed.  Either the numerical   
              or symbolical value may be used.
